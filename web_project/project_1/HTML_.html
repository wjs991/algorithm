HTML Tag 정리

기본 구조

  • <!DOCTYPE>
    : 마크업 언어용 DTD 태그이다. HTML5는 다음과 같이 써준다.
  • <HTML>
    문서의 속성을 지정하기 위한 태그. 이 태그안에 타이틀이나 메타 태그를 넣는다.
  • <body>
    HTML 문서의 본 모양을 나타내기 위한 태그.
  • <title>
    HTML의 제목을 선언하는 태그.
  • <meta>
    H부가 정보를 선언하는 태그..
  • <charset>
    인코딩을 선언하는 태그로 HTML5에서 추가되었다. 이전까지는 태그로 선언했었다.
  • <link>
    별도의 CSS 선언 등을 연결해주는 태그.

텍스트 관련

  • <h>
    제목(heading)을 표시할 때 사용된다. <h1>이 가장 크고 <h6>이 가장 작다. 크기는 브라우저마다 표시하는 방법이 달라 다르게 나타날 수 있다. CSS를 쓰면 크기, 색상, 폰트 등을 변경할 수 있다.
  • <p>
    문단을 열어주는 태그. 이 태그를 사용하면 기본적으로 문단 하단에 약 1줄 가량의 빈 공간이 생기기 때문에 <br> 을 써주는 경우가 많은데, 문단이라면 <p> 태그로 묶어주는 것이 올바른 사용 방법이다. 이는 css로 지정하여 없앨 수 있다.
  • <b>
    두껍게(bold) 효과를 준다.
  • <i>
    텍스트 italic꼴로 표시
  • <strong>
    포함된 텍스트를 강조할때 사용한다.
  • <em>
    약하게 강조할때 사용한다.

목록

  • <ul>
    순서가 없는 목록을 표시한다. 현재 필자가 사용하는 태그이다.
  • <li>
    목록에서 각 항목은 <li>와</li>사이에 넣는다.
  • <ol>
    순서가 있는 목록을 표시
  • <dl>
    정의 목록을 표시
  • <dt>
    정의 목록의 뜻풀이를 기술한다.

링크, 이미지

  • <a>
    하이퍼링크를 생성하는 태그이다. href 속성을 써서 작성한다.a와 a는 곂쳐서 사용하지 않는다.
    • 전북대학교
  • <img />
    이미지를 추가해주는 태그
    • src: 이미지 파일의 경로를 지정한다.
    • alt: 이미지를 볼 수 없을 경우 이미지에 대한 설명을 제공, 이미지가 출력되면 표기되지는 않는다
    • title: 이미지에 대한 추가 정보를 제공한다.
    • height,width: 이미지의 세로,가로폭을 지정
  • <svg>
    페이지에 svg형신의 그래픽을 추가해준다.
  • <progress>
    페이지에 진행 상황 막대를 추가하는 태그이다.

테이블

  • <table>
    테이블을 만드는 태그
  • <tr>
    행을 시작한다.
  • <td>
    셀을 표현한다.
    • 테 스 트
  • <th>
    테이블의 행, 열의 머리말을 나타냄

    • 날짜 수입 수출
    1월 1일 100,000 100,000

  • colspan, rowspan 속성으로 병합할 수 있다.

    • 날짜 수입 수출
    1월 1일 100,000

  • <caption>
    테이블의 제목을 기술한다.
  • <colgroup>,<colgroup>
    테이블 상단에 넣어 테이블의 열 정보를 기술. 셀의 너비를 지정할 때 쓰인다.
  • <thead>,<tbody>,<tfoot>
    테이블의 세부 구조를 기술 한다. 각각 상단 본문 하단

폼

CGI(PHP, ADP, JSP 등)와 상호작용하기 위해 입력값을 받는 역할
<form> 태그 - *폼 만들기*
└ 폼을 만드는 가장 기본적인 태그. <form> 태그와<form> 태그 사이에 여러 폼요소와 관련된 태그를 넣는다.
└ <form> 태그 형식
<form 속성="속성값"> 여러 폼 요소 </form> ← <form> 태그의 속성 형식

[폼 태그의 속성]

1. method 폼을 전송할 방식을 선택하는 속성. 사용할 수 있는 값은 get과 post 로서 get은 256 ~ 4096 바이트의 정보만을 서보로 넘길 수 있다.하지만 post는 사용자의 입력 내용의 길이에 제한을 받지 않는다. 그래서 대개 post를 사용한다. ex) <form method="post"></form>

2. name 한 문서 안에 여러 개의 폼이 있을 수 있기 때문에 폼을 식별하기 위한 폼의 이름.

3. action 폼을 전송할 서버 쪽의 스크립트 파일을 지정.

4. target 현재 창이나 다른 창에서 열릴지 새창의 위치를 정해준다.
  • <form>
    입력값을 받는 영역을 지정한다. 한 페이지에 여러 개가 들어갈 수는 있지만, 영역 안에 영역이 들어가는 것은 불가능하다.
  • <input>
    입력값 요소 지정. type에 따라 다른 입력값을 취함
    • <text> : 한 줄 짜리 문자열 값
    • <number> : 숫자
    • <url> : 도메인 주소
    • <email> : 이메일
    • <tel> : 전화번호
    • <search> : 검색어
    • <range> : 지정한 범위의 숫자
    • <color> : 색
    • <date> : 날짜
    • <time> : 시간
    • <datetime> : 날짜 + 시간
    • <checkbox> : 체크박스
    • <radio> : 라디오 버튼
    • <button> : 버튼
    • <submit> : 누를 경우 입력값 전송
    • <reset> : 누를 경우 입력값 초기화
    • <hidden> : 입력값을 수정하지 않고 보낼때 사용. 소스를 열어보지 않으면 사용자에게는 보이지 않는다.
    • <file> : 파일 업로드에 쓰임
  • <textarea>
    여러 줄의 문자열 값을 받는다.
  • <output>
    input태그 처럼 입력값이지만 수정할 수 없다.
  • <datalist>
    검색어에 들어갈 목록을 지정한다.
  • <select>
    선택 목록
    • <option> : 선택 목록에 들어갈 항목, radio와 비슷하다.
  • <legend>
    범례를 지정한다.
  • <label>
    해당하는 요소의 이름을 지정

기타

  • <iframe>
    페이지 안에서 또 페이지를 볼수 있다. 예를 들면 문서에서 구글 지도, youtube 등을 보는 것
    • <src> : 넣을 페이지의 주소를 지정한다.
    • <height>,<width> : 프레임의 높이, 너비를 지정한다.
    • <frameborder> : 프레임의 테두리 표시 여부를 지정한다. 0은 표시하지 않음, 1은 표시
    <iframe width="560" height="315" src="https://www.youtube.com/embed/a2GujJZfXpg" frameborder="0" allowfullscreen></iframe>
  • <object>
    웹 페이지 또는 외부 프로그램을 임베드 한다.
  • <style>
    css를 사용한다.
  • <script>
    자바스크립트나 vb스크립트를 사용
