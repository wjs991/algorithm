HTML5란

HTML은 인터넷상에서 표현하기 위한 기본 프로그래밍 언어이다. 이 HTML은 여러 버전이 존재해 왔었다.
현재 최신인 HTML5에 대해 알아 보겠다.

HTML5의 기본구조는 기존HTML과 다른 점은 없다.
하지만 기존 표현 보다 기능이 다양해졌다.

HTML5는 기존 액티브x와 같은 플러그인 기반의 각종 프로그램을 별도로 설치할 필요가 없이 영상, 음악을 손쉽게 볼수 있다. 이미 많은 모바일 서비스 영역에서 보편적 기술로 사용되고 있다.

HTML의 특징

  • 간단한 DTD선언 <!DOCTYPE>을 지원한다.
  • 기존 HTML문서와 호환가능
  • 편리한 폼 기능을 제공하여 유효성 검증 시간을 절약
  • 시멘틱 웹 기술을 지원하여 의미 있는 문서 구조 표현
    •     시멘틱 웹 기술: 사람이 인식하는 정보를 기계도 인 식할 수 있도록 하는 기술

    예제

    <!DOCTYPE html>
    <html lang="ko">
    <head>
      <meta charset="utf-8">
      <title>[페이지 제목]</title>
    </head>
    <body>
      <header>
        <h1>[사이트 제목]</h1>
        <h2>[사이트 부제목]</h2>
        <nav>
          <ul>
            <li>[메뉴1]</li>
            <li>[메뉴2]</li>
            <li>[메뉴3]</li>
          </ul>
          </nav>
      </header>
      <section>
        <article>
          <p>[본문 내용]</p>
        </article>
      </section>
      <aside> [사이드바 내용] </aside>
      <footer> <address>[주소 내용]</address> </footer>
    </body>
    </html>
